# AI_Map_Coloring
Final project for the Artificial Intelligence course at Kent State University in the Fall 2025 semester. A map coloring project that ensures no two bordering counties share a color using constraint satisfaction.

* County adjacency list obtained from: https://www.census.gov/geographies/reference-files/time-series/geo/county-adjacency.html

* County geospatial data obtained from: https://public.opendatasoft.com/explore/dataset/georef-united-states-of-america-county/export/?flg=en-us&disjunctive.ste_code&disjunctive.ste_name&disjunctive.coty_code&disjunctive.coty_name
